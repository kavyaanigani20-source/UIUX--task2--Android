# UI/UX Task 2 Android Implementation

This folder contains Android XML UI code for the Day 6-12 UI/UX task.

## What is included

- XML layouts for login, dashboard, profile, and settings screens
- Activity navigation using simple Android intents
- Buttons, text fields, cards, and screen navigation
- Centralized colors, strings, dimensions, and styles
- Responsive layout for tablet screens using `layout-sw600dp`
- LinkedIn post draft for showcasing the UI/UX process

## How to use

1. Create a new Android Studio project with an Empty Activity.
2. Use package name `com.example.uiuxtask2`, or update the package names in the Java files.
3. Copy the `app/src/main` folder from this deliverable into your project.
4. Run the project on a phone and tablet emulator to verify responsiveness.

## Screens

1. `MainActivity` - welcome/login screen
2. `DashboardActivity` - main navigation screen
3. `ProfileActivity` - profile details screen
4. `SettingsActivity` - preferences/settings screen
